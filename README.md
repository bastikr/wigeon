# Wigeon

A simple 2D/3D geometry package.

Geometric objects:

  * Vector
  * Point
  * Circle
  * Rectangle
  * LineSegment
  * Line
  * Ray
  * Polygon

Operations:

  * Rotation
  * Translation
  * Distances
  * Intersections
  * Inclusion
  * Winding number
  * Polygon offsetting
